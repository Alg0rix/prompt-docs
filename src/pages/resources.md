# Prompt Engineering Resources

This page contains a curated collection of resources to help you deepen your understanding of prompt engineering and large language models.

## Books and Publications

- **[Prompt Engineering Guide](https://www.promptingguide.ai/)** - Comprehensive guide by DAIR.AI
- **[Building LLM Applications](https://github.com/openai/openai-cookbook)** - OpenAI's cookbook for building LLM applications
- **[Natural Language Processing with Transformers](https://www.oreilly.com/library/view/natural-language-processing/9781098103231/)** - Book on transformer models
- **[Designing Prompts for LLMs](https://www.amazon.com/Designing-Prompts-Large-Language-Models/dp/1098146255)** - A practical guide to prompt engineering

## Tools and Frameworks

- **[LangChain](https://python.langchain.com/)** - Framework for developing applications powered by language models
- **[LlamaIndex](https://www.llamaindex.ai/)** - Data framework for LLM applications
- **[Guardrails](https://github.com/guardrails-ai/guardrails)** - Framework for validating LLM outputs
- **[LMQL](https://lmql.ai/)** - Query language for LLMs

## Academic Research

- **[Prompt Engineering Survey](https://arxiv.org/abs/2107.13586)** - Academic survey of prompt engineering techniques
- **[Chain of Thought Prompting](https://arxiv.org/abs/2201.11903)** - Google Research paper on reasoning through chain-of-thought
- **[ReAct Prompting](https://arxiv.org/abs/2210.03629)** - Paper on reasoning and acting in language models

## Communities and Forums

- **[r/PromptEngineering](https://www.reddit.com/r/PromptEngineering/)** - Reddit community for prompt engineers
- **[HuggingFace Forums](https://discuss.huggingface.co/)** - Community discussions on LLMs and NLP
- **[AI Alignment Forum](https://www.alignmentforum.org/)** - Discussions on alignment and safety in AI systems

## Online Courses

- **[Prompt Engineering for ChatGPT](https://www.deeplearning.ai/short-courses/chatgpt-prompt-engineering-for-developers/)** - Short course by DeepLearning.AI
- **[Building Systems with the ChatGPT API](https://www.deeplearning.ai/short-courses/building-systems-with-chatgpt/)** - Course on building LLM applications
- **[Generative AI with Large Language Models](https://www.coursera.org/learn/generative-ai-with-llms)** - Comprehensive course on Coursera
