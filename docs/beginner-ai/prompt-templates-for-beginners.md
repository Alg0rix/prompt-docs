# Prompt Templates for Beginners

If you are not sure what to type, start with a template.

These prompts are designed to be easy to copy, paste, and customize.

## Email Template

> Help me write a **[type of email]**. The situation is **[context]**. The tone should be **[friendly / professional / polite]**. Keep it under **[length]**.

Example:

> Help me write a follow-up email after a job interview. The tone should be professional and warm. Keep it under 120 words.

## Rewrite Template

> Rewrite this text so it sounds **[clearer / more professional / friendlier / simpler]**: **[paste text]**

## Summary Template

> Summarize this in **[3 bullet points / plain English / short paragraphs]**: **[paste text]**

## Study Help Template

> Explain **[topic]** to a beginner. Use simple language and give one example. Then give me 3 quiz questions.

## Resume Template

> Improve these resume bullet points so they sound clearer and stronger, but keep them honest and realistic: **[paste bullet points]**

## Small Business Template

> I run a **[type of business]**. Give me **[number]** content ideas for **[Instagram / email / flyers / website]**. Keep them simple and easy to use.

## Planning Template

> Create a simple plan for **[goal]**. I have **[time / budget / limits]**. Put the answer in a step-by-step list.

## Travel Template

> Plan a **[number]-day** trip to **[place]** for a **[type of traveler]**. Focus on **[budget / food / sightseeing / walking]**.

## Comparison Template

> Compare **[option A]** and **[option B]** for **[goal]**. Use a table and include pros, cons, and who each option is best for.

## Fresh Chat Handoff Template

> I am starting a new chat. Here is the context: **[short summary]**. My goal is **[goal]**. Please continue by **[specific task]**.

## Tips for Using Templates

- Replace the brackets with real details
- Keep only the parts you need
- Add audience, tone, and format when useful
- If the answer is weak, ask for a revision
